PinkElefant PEAR channel server

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover jbroadway.github.com/elefant
Listing available packages: 
pear remote-list -c elefant
Installing a package: 
pear install elefant/package_name
Installing a specific version/stability: 
pear install elefant/package_name-1.0.0
pear install elefant/package_name-beta
Receiving updates via a feed: 
http://jbroadway.github.com/elefant//feed.xml

PinkElefant - Simple, secure, and fast MVC web framework.

PinkElefant is a very simple, secure and fast MVC web framework for PHP. It re-imagines what elegant MVC in PHP can be by doing things the "PHP way" throughout and not just translating concepts awkwardly from other languages.

  • Package name: PinkElefant
  • License: MIT
  • Maintainers: Johnny Broadway (as lead)
  • Releases: 0.8.0 (beta)
  • Install command: pear install elefant/PinkElefant
